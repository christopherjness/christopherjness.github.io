DENSE SUSPENSIONS LAMMPS TUTORIAL



Welcome... This tutorial assumes you are using Mac or Linux and that you have a working knowledge of the terminal.

    Get started

  1. Read the tutorial here: https://www.overleaf.com/read/jfjbbhhfbgtc

    2. Download and compile LAMMPS

  2. Download a LAMMPS distribution (I used: LAMMPS Stable 5th June 2019)
    [Click here to download the .zip]
    [Click here to download the .tar.gz]
  3. Download the lubrication .cpp file file and .h file and place these in the directory: lammps-stable_5Jun2019/src/
  4. Enter the directory: lammps-stable_5Jun2019/src/STUBS/ and issue the command: make

    5. Generate packing at prescribed volume fraction $\phi$

  5. Input script to generate a particle packing: in.create
  6. Download a video of in.create simulation

    7. Run a rheology simulation

  7. Input script to shear the particle packing: in.run


Animations are generated using Ovito. Always refer to the LAMMPS website. Contact me: chris.ness@ed.ac.uk.
