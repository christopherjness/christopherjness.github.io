DENSE SUSPENSIONS LAMMPS TUTORIAL




    Get started

  1. Familiarise yourself with the physics of dense suspensions by reading e.g. Guazzelli and Pouliquen
  2. Next read the tutorial here. Other pedagogic articles are available
  3. The following assumes you are using Mac or Linux and that you have a working knowledge of the terminal.

    4. Download and compile LAMMPS

  4. Download a LAMMPS distribution (I used: LAMMPS Stable 5th June 2019)
    [Click here to download the .zip or here to download the .tar.gz]
  5. Using the Terminal, enter the directory: lammps-stable_5Jun2019/src/
  6. Issue the command: make yes-GRANULAR
  7. Issue the command: make yes-COLLOID
  8. Download the lubrication .cpp file file and .h file and place these in the directory: lammps-stable_5Jun2019/src/
  9. Enter the directory: lammps-stable_5Jun2019/src/STUBS/ and issue the command: make
  10. In my version of Mac (10.15.5) it is necessary to change the compiler settings by entering the directory: lammps-stable_5Jun2019/src/MAKE/
    and issuing the command: sed -i '' 's/g++/g++-9/g' Makefile.serial
  11. LAMMPS can then be compiled by entering the directory: lammps-stable_5Jun2019/src/ and issuing the command: make serial
  12. If successful, you can view the installed packages by running: ./lmp_serial -h

    13. Generate packing at prescribed volume fraction

  13. Download the input script to generate a particle packing: in.create and place it in the same directory as lammps-stable_5Jun2019 is in
  14. Issue the command: lammps-stable_5Jun2019/src/src/lmp_serial -in in.create
  15. Download a video of in.create simulation

    16. Run a rheology simulation

  16. Download the input script to shear the particle packing: in.run and place it in the same directory as lammps-stable_5Jun2019 is in
  17. Issue the command: lammps-stable_5Jun2019/src/src/lmp_serial -in in.run
  18. Download a video of in.run simulation
