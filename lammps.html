DENSE SUSPENSIONS LAMMPS TUTORIAL



Get started

  1. Familiarise yourself with the physics of dense suspensions by reading e.g. this or this.
  2. This site accompanies the tutorial here.
  3. The following assumes you are using Mac or Linux and that you have a working knowledge of the terminal.

I already have LAMMPS

  1. Put the lubrication .cpp file and .h file in your lammps/src/ directory.
  2. Install the granular and colloid packages by running in lammps/src/: make yes-GRANULAR yes-COLLOID
  3. Recompile LAMMPS
  4. Run the input scripts in.create and in.run just like any other.

I don't already have LAMMPS

  1. Download a LAMMPS distribution (I used: LAMMPS Stable 5th June 2019, download the .tar.gz here)
  2. Download the lubrication .cpp file and .h file and place these in the directory: lammps-stable_5Jun2019/src/
  3. Download the input scripts in.create and in.run and place them in the same directory as lammps-stable_5Jun2019 is in
  4. In Terminal, enter the directory: lammps-stable_5Jun2019/src/
  5. Issue the command: make yes-GRANULAR yes-COLLOID

Then do:

Compile and run serial LAMMPS on a laptop (I used MacOS 10.15.5)

  1. Ensure that lammps-stable_5Jun2019/src/MAKE/Makefile.serial addresses the correct compiler settings for your machine.
    (Probably not necessary on Linux, but on Mac I did: sed -i '' 's/g++/g++-11/g' lammps-stable_5Jun2019/src/MAKE/Makefile.serial, but see the documentation)
  2. Compile LAMMPS by entering the directory: lammps-stable_5Jun2019/src/ and issuing the command: make serial
  3. If successful, you can view the installed packages by running: ./lmp_serial -h

    4. Generate packing at prescribed volume fraction

  4. Issue the command: lammps-stable_5Jun2019/src/lmp_serial -in in.create
  5. The output data.file contains the starting configuration to be used by in.run

    6. Run a rheology simulation

  6. Issue the command: lammps-stable_5Jun2019/src/lmp_serial -in in.run

Or:

Compile and run parallel LAMMPS on a cluster (I used Eddie)

  1. Ensure the appropriate compiler is available. On Scientific Linux I ran: module load openmpi/1.10.1
  2. Compile LAMMPS by entering the directory: lammps-stable_5Jun2019/src/ and issuing the command: make mpi
  3. If successful, you can view the installed packages by running: ./lmp_mpi -h

    4. Generate packing at prescribed volume fraction

  4. Issue the command: mpirun -np 2 lammps-stable_5Jun2019/src/lmp_mpi -in in.create
  5. The output data.file contains the starting configuration to be used by in.run

    6. Run a rheology simulation

  6. Issue the command: mpirun -np 2 lammps-stable_5Jun2019/src/lmp_mpi -in in.run
